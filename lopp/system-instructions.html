System Instructions

How to use EDUPATH+ effectively.

1) Start with My Courses

  • Enroll in your courses for your department and part.
  • Admins can assign official modules per department/part.

2) Resource Hub

  • Upload notes/past papers. Save useful docs for later.
  • Use View 👁️ / Download 📥 / Save 💾 buttons.

3) Q&A

  • Ask questions, like/dislike once, and comment (toggle comments like YouTube).
  • Top interactions appear first in your department.

4) Chat

  • Join rooms or DM people privately via Inbox.

5) Privacy & Security

  • Private DMs are visible only to the two participants (RLS enforced).
  • Admins cannot read DMs unless they are part of the chat.
